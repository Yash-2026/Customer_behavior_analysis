# 👨🏻‍💻 Customer Behavior Analysis | Data Analytics Portfolio Project

This project demonstrates a complete end-to-end data analytics workflow using Python, SQL, and Power BI. It showcases the process of cleaning, analyzing, and visualizing customer shopping data to generate meaningful business insights that support data-driven decision-making.

## 💡 Key Insights

- Clothing was the top-performing product category.
- The average purchase amount was approximately **$59.76**.
- Customer purchasing behavior varied across different age groups and genders.
- Most customers preferred Card and Cash payment methods.
- The Power BI dashboard helped visualize customer trends and business performance effectively.

## 🚀 How to Run

### 1. Clone the Repository

```bash
git clone https://github.com/Yash-2026/Customer_behavior_analysis.git
```

### 2. Open the Project

Open the `Customer behavior analysis.ipynb` file in **Jupyter Notebook** or **JupyterLab**.

### 3. Install Required Libraries

Install the required Python packages before running the notebook.

```bash
pip install pandas numpy matplotlib seaborn sqlalchemy pymysql
```

### 4. Run the Notebook

Execute all cells in the notebook to:
- Load the dataset
- Clean and preprocess the data
- Perform Exploratory Data Analysis (EDA)

### 5. Import Data into MySQL

- Create a MySQL database.
- Import the cleaned dataset into MySQL.
- Run the SQL queries available in `customer_behaviour.sql`.

### 6. Open the Power BI Dashboard

Open the `customer_behaviour Dashboard.pbix` file using **Power BI Desktop** to explore the interactive dashboard and business insights.
